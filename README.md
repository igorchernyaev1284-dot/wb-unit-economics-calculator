# Wildberries Unit Economics Calculator (n8n Python Node)

Автоматизированный скрипт на Python для расчета юнит-экономики селлеров Wildberries. Скрипт разработан специально для интеграции в n8n (внутри ноды Code) и агрегирует сырые финансовые отчеты маркетплейса.

### 💼 Бизнес-ценность решения:
* **Исключение ручного труда:** Полностью заменяет ручной разбор еженедельных Excel-отчетов Wildberries финансовым аналитиком или менеджером.
* **Предотвращение кассовых разрывов:** Мгновенно подсвечивает нерентабельные товары (SKU с отрицательной маржой), которые сливают бюджет на логистику и возвраты.
* **Высокая скорость:** Обрабатывает сотни строк транзакций за миллисекунды прямо внутри автоматического сценария.

### ⚙️ Как это работает в n8n-сценарии:
1. **Входные данные:** n8n забирает еженедельный финансовый отчет Wildberries (через API или из Google Таблицы).
2. **Обработка (этот скрипт):** Скрипт группирует транзакции по артикулам (SKU), сопоставляет их с себестоимостью (COGS), рассчитывает общую логистику, возвраты, выплаты и чистую прибыль.
3. **Выходные данные:** Скрипт отдает чистый JSON-массив с агрегированными показателями маржинальности по каждому товару для отправки в Telegram-чат руководителя или BI-дашборд.

### 📊 Пример структуры выходных данных (JSON):

```json
  "cogs_per_unit": 300,

  "returns_qty": 2,

```
